# Backlog: Beide Temperaturen im Vergleich anzeigen

## Status
- [ ] Todo
- Created: 2026-01-15
- Priority: Medium

## Beschreibung

Im Vergleichs-Report beide Temperaturwerte anzeigen:
- **Echte Temperatur** (Lufttemperatur)
- **Gefühlte Temperatur** (Feels-like / Windchill)

## Akzeptanzkriterien

- [ ] Vergleichs-Tabelle zeigt beide Temperaturen
- [ ] Klare Beschriftung: "Temp" und "Gefühlt"
- [ ] Beide Werte in °C
- [ ] Format: z.B. "-2°C (gefühlt -8°C)"

## Betroffene Komponenten

- Comparison Engine
- Report Formatter
- Email Templates

## User Story

Als Weitwanderer möchte ich sowohl die echte als auch die gefühlte Temperatur sehen, damit ich meine Kleidung besser planen kann.
